Overview

This model grasp the intent for a given sentence and classifies whether it is yes/no, alternative, wh-questions, prohibitions, requirements, or strong requirements. Try using the KLUE-RoBERTa-base_sae model as a KLUE-RoBERTa-base model trained on kor_sae dataset.

Huggingface: klue-roberta-base_sae
Dataset: kor_sae
Open API: On Ainize

Notice

  • Type the sentence you want to know the intent and click the Grasp sentence intent button.
  • This model trained with Korean dataset.
  • You can use English, but don't expect accuracy.
